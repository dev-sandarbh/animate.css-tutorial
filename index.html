So, then Animate.css makes it easy and fun for us to make our content interactive with the help of predefined animations.

So, i have used this css library just for practice and had tried to animate some elements and divs. Here you go. Feel free to share your valuable ideas and comments. Enjoy article.

My Experience with Animate.css

I really enjoyed using this free css library.But in my opinion it has some limitations as well....And those limitations are:-

  1. Every animation occurs only when page reloads and i think you won't love to load a webpage everytime just to see that animation....I mean who does that?
  2. Since every animation loads only once when page loads so you won't be able to see elements which are at the bottom of the page; so no point in using animations for those elements.
  3. Also, it is not good to give every element an infinite animation-duration just to see all the animations applied to all elements. It is frivolous!!!
  4. Moreover it has pre-defined animations which you would have to use. What if you want to have animation of your own? Then DIY it.
  5. Also if you are using only one or two animations, then it is not a good choice to load a 97KB file everytime.

So, if you want custom animations and flexibility at your will, you need write your own code/styles for that. And i think that is the best option as well.But it is good for getting hands dirty with CSS-Animations for stating. So go ahead and start writing your own code.
